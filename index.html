The Briefing Process
Briefing
0 / 3
  • All design requests follow a standardised brief template — no verbal or ad-hoc asks.
  • Project goals and target audiences are clearly defined for every request before work begins.
  • All necessary assets — copy, logos, images — are provided upfront before a designer starts.
2
The Feedback Loop
Feedback
0 / 3
  • All feedback is consolidated in one place — not scattered across emails, Slack threads, or Word docs.
  • Feedback is specific and actionable — vague comments like "make it pop" are not accepted.
  • There is a clear, agreed limit on revision rounds to prevent scope creep and endless changes.
3
File Management & Brand Consistency
Brand
0 / 3
  • You can instantly find the final, approved version of any past design asset — no version confusion.
  • A central, accessible brand library exists for all logos, fonts, colour codes, and approved imagery.
  • You are confident that all public-facing materials — across every channel — are 100% on-brand.
4
Tools & Resources
Canva Trap
0 / 4
  • Non-designers are not spending more than 3 hours/week on "simple" design tasks in Canva or similar tools.
  • You do not rely on expensive, ad-hoc freelancers for projects that exceed in-house tool capabilities.
  • Campaigns have never been delayed because of a design bottleneck or technical limitation.
  • You are not manually resizing the same design for different social media platforms and formats.
5
Team & Process Ownership
Process
0 / 2
  • A single person owns the design workflow — requests, priorities, and delivery do not get lost between stakeholders.
  • Your team has a predictable, reliable design output — not a bottleneck that blocks campaign launches.
